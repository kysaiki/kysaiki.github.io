    Flushr

    Java, HTML, CSS, MySQL, AWS Elastic Beanstalk, RDS, S3, Route 53

    Flushr is a webapp that searches for restrooms in the area and sorts the results based on different settings. Users are able to leave reviews and comments about restrooms and rate restrooms based upon certain criteria. This app utilized AWS Elastic Beanstalk to host it as well as AWS RDS to store the data.

    Grouplan

    Swift, Google API, Firebase

    Grouplan is an iOS application that allows users to plan out detailed events together from different devices. Users can create events, view the events they are added to, or join new events via an event key and password. Firebase updates user's changes automatically, allowing users to make changes together in real time

    COVID Stock Correlation

    Python Flask, HTML, CSS, AWS S3, AWS EC2

    COVID Stock Correlation is a webapp finds the correlation between COVID cases in a state and a stock's price. Users input a stock ticker symbol and a state abbreviation and the website will display a dotplot of cases mapped with stock price and the correlation strength.
